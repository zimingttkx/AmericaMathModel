# AI 可视化生成提示词模板

> **使用方法**: 复制相应模板，替换【】中的内容，发送给AI

## 📊 场景1：算法性能对比（最常用）

### 提示词模板
```
请帮我生成一个算法性能对比图：

【基本信息】
图表类型：分组柱状图
数据内容：{算法数量}个算法在{数据集数量}个数据集上的准确率

【数据】
算法名称：{列出算法名}
数据集名称：{列出数据集名}
准确率数据：
算法1：{数值1, 数值2, ...}
算法2：{数值1, 数值2, ...}
算法3：{数值1, 数值2, ...}

【样式要求】
- 整体风格：[Nature风格/Science风格/IEEE风格/比赛风格]
- 图表尺寸：{宽}x{高}英寸
- 配色方案：[学术配色/高对比度/自定义]
- 是否显示网格：[是/否]
- 图例位置：[右上/左上/右下/左下]

【标注要求】
- 标题："{图表标题}"
- X轴标签："{X轴标题}"
- Y轴标签："{Y轴标题}"
- Y轴范围：{最小值}到{最大值}
- 是否显示数值：[是/否]
- 数值格式：{保留小数位数}

【输出要求】
- 保存路径：{figures/xxx.png}
- 分辨率：{300/600} DPI
- 文件格式：[PNG/PDF/SVG]

请使用以下代码模板：
from visualization.matplotlib_templates.bar_charts import plot_grouped_bar_chart
```

### 实际示例
```
请帮我生成一个算法性能对比图：
```
