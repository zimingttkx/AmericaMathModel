# 可视化快速参考卡

## 🎯 30秒决策流程

```
1. 数据是什么类型？
   时间序列 → 折线图
   分类对比 → 柱状图
   相关性 → 热力图/散点图
   分布 → 直方图/箱线图
   占比 → 饼图/堆叠图

2. 用在哪里？
   发表论文 → Nature风格（简约、600DPI）
   参加比赛 → 比赛风格（清晰、300DPI）
   汇报演示 → 演示风格（鲜艳、150DPI）

3. 直接复制提示词模板给AI！
```

## 📊 常用图表速查

| 场景 | 图表 | 风格 | 提示词模板 |
|------|------|------|-----------|
| 算法对比 | 分组柱状图 | 比赛风格 | 场景1 |
| 趋势变化 | 折线图 | Nature风格 | 场景2 |
| 相关性 | 热力图 | Science风格 | 场景3 |
| 数据分布 | 箱线图 | 比赛风格 | 场景4 |
| 预测评估 | 散点图 | Nature风格 | 场景5 |
| 综合结果 | 多面板图 | Nature风格 | 场景6 |

## 🔧 配置速查

```python

    'figsize': (3.5, 2.5),
    'dpi': 600,
    'font.size': 6,
    'linewidth': 1.0,
    'grid': False
}
```
