```
│       - 带不确定性：带置信区间的折线图
│
├─ 分类对比（多个类别比较）
│   └─→ 使用柱状图
│       - 单组对比：简单柱状图
│       - 多组对比：分组柱状图
│       - 部分与整体：堆叠柱状图
│
├─ 相关性（变量间关系）
│   └─→ 使用散点图或热力图
│       - 两个连续变量：散点图
│       - 多个变量：热力图
│       - 预测评估：预测vs实际散点图
│
├─ 分布（数据分布情况）
│   └─→ 使用直方图或箱线图
│       - 单变量分布：直方图
│       - 多组对比：箱线图
│       - 密度估计：小提琴图
│
└─ 占比（部分与整体）
    └─→ 使用饼图或堆叠图
        - 少量类别（<5）：饼图
        - 时间变化：堆叠面积图
```
