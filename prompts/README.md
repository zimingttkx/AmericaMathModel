## 🚀 三步快速使用流程

### 第1步：找到合适的Prompt文件

```
示例：我需要画一个算法对比图

操作：
1. 打开 prompts/01_可视化图表.md
2. 搜索"算法性能对比"或"分组柱状图"
3. 找到场景1
4. 复制完整的Prompt模板
```

### 第2步：填写【】内容

```
示例填写：
【算法名称】SVM, Random Forest, XGBoost
【数据集名称】Iris, Wine, Breast Cancer
【准确率】
SVM: 96.67, 98.31, 95.23
Random Forest: 97.22, 97.83, 96.54
XGBoost: 98.15, 98.92, 97.87
```

### 第3步：发送给AI

```
直接粘贴填写好的Prompt给AI（ChatGPT/Claude/其他）
获得完整的Python代码
复制到你的项目中运行
保存图表
完成！
```
